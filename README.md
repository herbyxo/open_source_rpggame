# open_source_rpggame - Software Process Improvement Demo Open Source Project

## Description
This is a 2D multiplayer memory combat game where 2-4 players compete using both strategy and memory. Each player starts with 3 characters on the battlefield. During gameplay, numbers will briefly appear on the screen, and players must memorize them. Correctly remembering these numbers can provide advantages such as gaining new characters, weapons, or special cards.
Players must balance tactical positioning, combat decisions, and memory skills to defeat their opponents.

## How to Play
1. The game is played by 2–4 players.
2. Each player begins with 3 characters on the battlefield.
3. Players take turns moving characters and performing combat actions.
4. During the match, numbers will briefly appear on the screen.
5. Players must memorize these numbers before they disappear.
6. Later in the round, players may be asked to recall the numbers.
7. Correct answers can reward players with:
- New characters
- Weapons
- Special cards
- Combat advantages
Players can use these rewards to strengthen their team and defeat opponents.

## Objective
The objective of the game is to defeat your opponents' characters using tactical combat and memory skills.
Players who correctly remember the numbers that appear during the game can gain powerful advantages such as:
- Unlocking new characters
- Equipping stronger weapons
- Using special cards
- Gaining tactical advantages during combat
The last player with surviving characters wins the match.
